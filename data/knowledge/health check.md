# 健康检查说明 | how to use PAR-Q

- 问卷仅用于初步筛查，不替代医生诊断。
- 有「是」的情况：
  - 计划生成时应自动降级为低强度建议，避免 HIIT / 大重量。
  - 鼓励用户先进行线下评估。
- 如合并 red_flags.md 中症状，应提示立刻就医而不是调整计划。

# Health Check Instructions: how to use PAR-Q

The questionnaire is only for preliminary screening and does not replace a doctor's diagnosis.
- Situations where there is a "yes" :
When the plan is generated, it should be automatically downgraded to a low-intensity recommendation to avoid HIIT/heavy weight.
Encourage users to conduct offline evaluations first.
If symptoms in red_flags.md are combined, immediate medical attention should be prompted instead of adjusting the plan.
